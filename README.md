
# StudySphere - Student Productivity Dashboard

**StudySphere** is a lightweight, client-side web application designed to assist students in managing daily academic tasks, tracking focus sessions, and organizing study notes. 

The application operates entirely within the browser, utilizing local storage for data persistence, eliminating the need for backend infrastructure or external databases.


## Features

* **Task Management:** Users can add, complete, and delete specific study tasks. Progress is tracked via a dynamic completion bar.
* **Focus Timer (Pomodoro):** A built-in timer featuring standard 25-minute work intervals and 5-minute breaks.
* **Study Analytics:** The application logs completed focus sessions and visualizes the accumulated study hours in a weekly bar chart.
* **Quick Notes:** A dedicated section to draft, store, and delete short academic notes.
* **Data Persistence:** All user inputs (tasks, notes, and timer statistics) are saved locally using the browser's `localStorage` API, ensuring data remains available between sessions.
